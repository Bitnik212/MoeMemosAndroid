<b>Moe Memos</b> is an app to help you capture thoughts and ideas. To use this app, you must have a self-hosted Memos server set up and running. Find out more at <a href="https://github.com/usememos/memos">https://github.com/usememos/memos</a>

Features:

- Write memos like tweeting to yourself
- Sync with your own Memos server
- Material You design with dynamic themes and themed icon
- Group memos with tags
- Pin and archive memos
- View your progress with a graph
- Full privacy protection, no data collection
